
Project Zero / 零計畫 — GitHub Pages 靜態站原型
=======================================

簡介
- 網站名稱：Project Zero / 零計畫
- 主色：薄荷綠
- 技術：純前端 (HTML/CSS/JS)；可部署至 GitHub Pages

快速開始
1. 將整個 project-zero 資料夾上傳到一個 GitHub repository。
2. 在 Repository 的 Settings -> Pages 中啟用 GitHub Pages（branch: main 或 gh-pages）。
3. 頁面會在幾分鐘內上線。

備註
- assets/js/marked.min.js 目前為簡易 stub；若要支援完整 Markdown 解析，可替換為官方 marked.min.js。
- 日記與 blog 可用靜態 .html 或 .md 放入對應資料夾。

結構
[
  "assets",
  "diary",
  "gallery",
  "index.html",
  "collections.html"
]

Enjoy — 我把原型先放在本地，你可以下載後直接上傳到 GitHub。
